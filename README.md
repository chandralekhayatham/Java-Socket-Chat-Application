# Java Socket Chat Application 💬

A desktop-based chat application developed using Java Socket Programming and Java Swing.

## Features

- Real-time Messaging
- Multi-client Support
- Java Swing GUI
- Socket Programming
- Multithreading
- Dark Mode
- Save Chat
- Clear Chat
- Message Timestamps
- User-friendly Interface

## Technologies Used

- Java
- Java Swing
- Socket Programming
- Multithreading
- OOP Concepts

## Project Structure

```
src
│
├── client
│   ├── ChatClient.java
│   └── ChatClientGUI.java
│
└── server
    ├── ChatServer.java
    └── ClientHandler.java
```

## How to Run

1. Run ChatServer.java
2. Run ChatClientGUI.java
3. Enter Username
4. Click Connect
5. Open another client and chat
